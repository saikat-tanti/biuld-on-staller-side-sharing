# 🚗 Ride Sharing Smart Contract (Soroban - Stellar)

## 📌 Project Description

This project is a decentralized ride-sharing smart contract built using Soroban on the Stellar blockchain. It enables riders to create ride requests and drivers to accept and complete rides without relying on centralized platforms.

The goal is to demonstrate how blockchain can power trustless, transparent ride-sharing systems.

---

## ⚙️ What It Does

The smart contract allows:

- Riders to create ride requests with pickup and destination details
- Drivers to accept ride requests
- Rides to be marked as completed
- Anyone to query ride details from the blockchain

All ride data is stored on-chain, ensuring transparency and immutability.

---

## ✨ Features

- 📍 Create Ride Requests
- 🚖 Driver Ride Acceptance
- ✅ Ride Completion Tracking
- 🔍 View Ride Details
- 🔐 Decentralized & Trustless
- ⛓️ Built on Stellar Soroban Smart Contracts
